Ping:
sudo python3 NetworkApplications.py ping google.com --count 4 --timeout 2

Traceroute:
sudo python3 NetworkApplications.py traceroute google.com --protocol icmp  --timeout 2
sudo python3 NetworkApplications.py traceroute google.com --protocol udp --timeout 2

Multithreaded Traceroute:
sudo python3 NetworkApplications.py mtroute google.com --protocol udp --timeout 2
sudo python3 NetworkApplications.py mtroute google.com --protocol icmp --timeout 2

Web server:
python3 NetworkApplications.py web --port 8080
http://127.0.0.1:8080/index.html

Proxy:
python3 NetworkApplications.py proxy --port 8000
curl -x 127.0.0.1:8000 google.com
